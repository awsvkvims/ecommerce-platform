# Enterprise Ecommerce Platform

A full-stack ecommerce application using:
- React
- NestJS
- PostgreSQL
- AWS
- Terraform
- GitHub Actions


## Planned Architecture

- Frontend: React (S3 + CloudFront)
- Backend: NestJS (ECS Fargate)
- Database: PostgreSQL (RDS)
- CI/CD: GitHub Actions
- IaC: Terraform
